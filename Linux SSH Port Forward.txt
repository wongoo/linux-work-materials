ssh 隧道
--------

# 本地转发


ssh -L 2121:host1:21 host2
	---> 通过 host1:2121 访问 host2:21 端口, 命令在host1上执行

范例：
ssh -L 8010:192.168.0.12:8080 192.168.0.11
ssh -L 8083:192.168.0.12:8080 root@192.168.0.11 -N
ssh -L 8082:localhost:8080 root@192.168.0.12 -N
ssh -L

=====================================================

# 远程转发

ssh -f -N -R 2121:host2:21 host1
	--> 通过 host1:2121 访问 host2:21, 在host2上执行

ssh -R 8081:192.168.0.12:8080 192.168.0.11
ssh -f -N -R 6790:10.104.113.160:22 root@47.254.89.87


sshpass -p 'test2018' ssh -f -N -R 6790:10.104.113.160:22 test@47.254.89.87
